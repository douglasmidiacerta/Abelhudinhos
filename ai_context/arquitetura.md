# Arquitetura - Abelhudinhos

## Visão Geral
Landing page e plataforma de presentes digitais personalizados. O sistema gera uma experiência tipo "Stories" para apresentar fotos, linha do tempo, curiosidades, e joguinhos para casais.

## Tecnologias Principais
- HTML5
- CSS (Vanilla, responsivo)
- JavaScript (Vanilla)

## Arquivos Principais
- `index.html`: Landing page (apresentação do produto, carrossel de features, depoimentos, preçário).
- `presente.html`: Possível demo/preview do presente.

## Estrutura de Estilos
Usa custom properties (`--p`, `--dark`, etc.) no `:root`.
Design limpo, moderno, voltado para conversão e impacto visual.

## Componentes Notáveis
- `nav`: Barra de navegação fixa com efeito blur.
- `hero`: Seção principal com corações flutuantes animados via JS.
- `feat-carousel`: Carrossel de recursos que simula a tela de celulares com animações para transição de slides e alteração de conteúdo.
- `pricing`: Exibição do valor de venda e CTA.
